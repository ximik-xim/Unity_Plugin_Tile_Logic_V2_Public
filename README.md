Плагин нужен для работы с картой состоящей из тайлов. 

Реализованы разные подходы к созданию тайловой карты, и соединения таилов друг за другом
- Просто спавн след. таила по триггеру
- Вычисление точки спавна след. таила через пересечения вектора направления
- Спавн след. таила, используя точку соединения на самом таиле

---------------------------------------------------------------------------------------------------------

Примеры карт с тайлами и инструкция находятся в папке Tile Logic V2 -> Example -> Example Scene Tile

Примеры способов движения таилов находятся в папке Tile Logic V2 -> Example -> Example Scene Run

Примеры с использованием Spline в папках
Tile Logic V2 -> Other Plagin -> Tile And Spline(Spline - Unity) -> Spline V1 -> Example Scene

Tile Logic V2 -> Other Plagin -> Tile And Spline(Spline - Unity) -> Tile and Spline V1 -> Example Scene

Tile Logic V2 -> Other Plagin -> Tile And Spline(Spline - Unity) -> Tile and Spline V2 -> Example Scene
